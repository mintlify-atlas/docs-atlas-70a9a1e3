## Overview

This Neovim configuration is built on top of [LazyVim](https://www.lazyvim.org/), a Neovim setup powered by lazy.nvim. It provides a modern, feature-rich development environment with minimal configuration overhead.

## Configuration Structure

The configuration follows LazyVim's modular structure:

```
nvim/
├── init.lua                 # Entry point
└── lua/
    ├── config/              # Core configuration
    │   ├── lazy.lua        # Plugin manager setup
    │   ├── options.lua     # Vim options
    │   ├── keymaps.lua     # Key mappings
    │   └── autocmds.lua    # Auto commands
    └── plugins/             # Plugin configurations
        ├── colorscheme.lua
        ├── obsidian.lua
        └── snacks-config.lua
```

## Plugin Manager Setup

The configuration uses [lazy.nvim](https://github.com/folke/lazy.nvim) for plugin management with automatic installation:

```lua
local lazypath = vim.fn.stdpath("data") .. "/lazy/lazy.nvim"
if not (vim.uv or vim.loop).fs_stat(lazypath) then
  local lazyrepo = "https://github.com/folke/lazy.nvim.git"
  vim.fn.system({ "git", "clone", "--filter=blob:none", "--branch=stable", lazyrepo, lazypath })
end
vim.opt.rtp:prepend(lazypath)
```

### Plugin Configuration

```lua
require("lazy").setup({
  spec = {
    { "LazyVim/LazyVim", import = "lazyvim.plugins" },
    { import = "plugins" },
  },
  defaults = {
    lazy = false,
    version = false,  -- Use latest git commit
  },
  install = { colorscheme = { "tokyonight", "habamax" } },
  checker = {
    enabled = true,   -- Auto-check for updates
    notify = false,   -- Silent updates
  },
})
```

<Note>
  LazyVim automatically lazy-loads plugins for optimal startup performance.
</Note>

## Custom Plugins

### Obsidian Integration

Seamless integration with Obsidian for note-taking:

```lua
return {
  "epwalsh/obsidian.nvim",
  version = "*",
  lazy = true,
  ft = "markdown",
  dependencies = {
    "nvim-lua/plenary.nvim",
    "hrsh7th/nvim-cmp",
  },
  opts = {
    workspaces = {
      {
        name = "personal",
        path = "/path/to/your/obsidian/vault",
      },
    },
    completion = {
      nvim_cmp = true,
      min_chars = 2,
    },
    new_notes_location = "notes_subdir",
    daily_notes = {
      folder = "/daily notes",
      date_format = "%m-%d-%Y",
      alias_format = "%B %-d, %Y",
      default_tags = { "daily-notes" },
    },
  },
}
```

<Warning>
  Update the workspace path to match your Obsidian vault location.
</Warning>

## Visual Customization

Custom visual selection highlight for better visibility:

```lua
vim.api.nvim_set_hl(0, "Visual", { bg = "#ffff00", fg = "#000000" })
```

This creates a bright yellow background with black foreground for selected text.

## Getting Started

1. Install Neovim 0.9.0 or later
2. Backup existing configuration
3. Clone or symlink this configuration to `~/.config/nvim`
4. Launch Neovim - plugins will install automatically
5. Restart Neovim after initial installation

<Note>
  LazyVim provides extensive [documentation](https://www.lazyvim.org/) and keybindings. Press `<space>` to see available commands.
</Note>

## Customization Tips

### Adding New Plugins

Create a new file in `lua/plugins/` directory:

```lua
return {
  "username/plugin-name",
  opts = {
    -- plugin options
  },
}
```

### Changing Color Scheme

Edit `lua/plugins/colorscheme.lua` and update the default:

```lua
{
  "LazyVim/LazyVim",
  opts = {
    colorscheme = "gruvbox",  -- Change to your preferred scheme
  },
}
```

### Custom Keymaps

Add mappings in `lua/config/keymaps.lua`:

```lua
vim.keymap.set("n", "<leader>xx", "<cmd>YourCommand<cr>", { desc = "Description" })
```
