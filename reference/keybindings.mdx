## Overview

This page provides a comprehensive reference of all keyboard shortcuts configured in this dotfiles setup. The keybindings are organized by application for easy lookup.

<Note>
All keybindings can be customized by editing their respective configuration files.
</Note>

## Tmux Keybindings

Tmux is configured with Vim-style keybindings and uses `Ctrl-Space` as the prefix.

### Prefix Key

| Key | Action | Description |
|-----|--------|-------------|
| `Ctrl-Space` | Prefix | Main prefix key (replaces default `Ctrl-b`) |

<Tip>
The prefix key must be pressed before any tmux command. For example, to enter copy mode, press `Ctrl-Space` then `[`.
</Tip>

### Copy Mode

| Key | Action | Description |
|-----|--------|-------------|
| `Prefix + [` | Enter copy mode | Start navigating and copying text |
| `v` | Begin selection | Start visual selection (in copy mode) |
| `y` | Copy selection | Copy selected text to clipboard (in copy mode) |
| `V` | Rectangle toggle | Toggle rectangular selection (in copy mode) |

### General Settings

- **Mouse support**: Enabled
- **Vi mode keys**: Enabled in copy mode
- **Base index**: Windows start at 1 (instead of 0)
- **Renumber windows**: Automatic when windows are closed

## AeroSpace Keybindings

AeroSpace is a tiling window manager for macOS. All keybindings use `Alt` (Option) as the modifier.

### Layout Management

| Key | Action | Description |
|-----|--------|-------------|
| `Alt + /` | Toggle tiles layout | Switch between horizontal and vertical tiles |
| `Alt + ,` | Toggle accordion layout | Switch between horizontal and vertical accordion |

### Focus Navigation

| Key | Action | Description |
|-----|--------|-------------|
| `Alt + h` | Focus left | Move focus to left window |
| `Alt + j` | Focus down | Move focus to window below |
| `Alt + k` | Focus up | Move focus to window above |
| `Alt + l` | Focus right | Move focus to right window |

<Tip>
These Vim-style navigation keys (hjkl) make it easy to move between windows without using the mouse.
</Tip>

### Window Movement

| Key | Action | Description |
|-----|--------|-------------|
| `Alt + Shift + h` | Move left | Move window to the left |
| `Alt + Shift + j` | Move down | Move window down |
| `Alt + Shift + k` | Move up | Move window up |
| `Alt + Shift + l` | Move right | Move window to the right |

### Window Resizing

| Key | Action | Description |
|-----|--------|-------------|
| `Alt + Shift + -` | Decrease size | Shrink window by 50 pixels |
| `Alt + Shift + =` | Increase size | Grow window by 50 pixels |

### Workspace Utilities

| Key | Action | Description |
|-----|--------|-------------|
| `Alt + Tab` | Back and forth | Toggle between current and previous workspace |
| `Alt + Shift + Tab` | Move to next monitor | Move workspace to next monitor (wraps around) |

### Service Mode

Service mode provides advanced window management operations.

| Key | Action | Description |
|-----|--------|-------------|
| `Alt + Shift + ;` | Enter service mode | Enter service mode for advanced commands |

#### Service Mode Commands

| Key | Action | Description |
|-----|--------|-------------|
| `Esc` | Reload config | Reload AeroSpace config and exit service mode |
| `r` | Reset layout | Flatten workspace tree and exit service mode |
| `f` | Toggle floating | Toggle between floating and tiling layout |
| `Backspace` | Close other windows | Close all windows except current |
| `Alt + Shift + h` | Join left | Join current window with left window |
| `Alt + Shift + j` | Join down | Join current window with window below |
| `Alt + Shift + k` | Join up | Join current window with window above |
| `Alt + Shift + l` | Join right | Join current window with right window |

<Note>
All service mode commands automatically return you to main mode after execution.
</Note>

### Additional Notes

<Note>
**Leader Key**: In LazyVim, the leader key is `Space` by default.

For example:
- `<leader>ff` means press `Space` then `f` twice
- `<leader>ca` means press `Space`, then `c`, then `a`
</Note>

<Tip>
Many keybindings in LazyVim use the **which-key** plugin, which shows available commands when you pause after pressing the leader key or another prefix.
</Tip>

## Customization

To customize keybindings, edit the respective configuration files:

- **Tmux**: `~/.config/tmux/tmux.conf`
- **AeroSpace**: `~/.config/aerospace/aerospace.toml`
- **Neovim**: `~/.config/nvim/lua/config/keymaps.lua`

<Note>
After modifying keybindings:
- Tmux: Reload with `tmux source-file ~/.config/tmux/tmux.conf`
- AeroSpace: Press `Alt + Shift + ;` then `Esc` to reload
- Neovim: Restart Neovim or run `:source ~/.config/nvim/lua/config/keymaps.lua`
</Note>
