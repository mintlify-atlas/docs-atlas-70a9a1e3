## Overview

This dotfiles configuration includes 151 Alacritty color themes from the official [alacritty-theme](https://github.com/alacritty/alacritty-theme) repository. You can easily switch between themes by importing them in your Alacritty configuration.

<Note>
To use a theme, import it in your `alacritty.toml`:
```toml
import = ["~/.config/alacritty/themes/themes/theme-name.toml"]
```
</Note>

## Popular Theme Families

<Tip>
Some themes come in multiple variants. Here are the most popular theme families:
</Tip>

### Catppuccin
A soothing pastel theme with four flavors:
- `catppuccin.toml` - Default variant
- `catppuccin_latte.toml` - Light variant
- `catppuccin_frappe.toml` - Medium variant
- `catppuccin_macchiato.toml` - Medium-dark variant
- `catppuccin_mocha.toml` - Dark variant

### Gruvbox
A retro groove color scheme:
- `gruvbox_dark.toml` - Classic dark
- `gruvbox_light.toml` - Classic light
- `gruvbox_material.toml` - Material variant
- `gruvbox_material_hard_dark.toml` - High contrast dark
- `gruvbox_material_hard_light.toml` - High contrast light
- `gruvbox_material_medium_dark.toml` - Medium contrast dark
- `gruvbox_material_medium_light.toml` - Medium contrast light

### GitHub
Official GitHub themes with accessibility variants:
- Dark: `github_dark.toml`, `github_dark_default.toml`, `github_dark_dimmed.toml`
- Light: `github_light.toml`, `github_light_default.toml`
- Colorblind: `github_dark_colorblind.toml`, `github_light_colorblind.toml`
- Tritanopia: `github_dark_tritanopia.toml`, `github_light_tritanopia.toml`
- High Contrast: `github_dark_high_contrast.toml`, `github_light_high_contrast.toml`

### Tokyo Night
A clean dark theme with three variants:
- `tokyo_night.toml` - Classic
- `tokyo_night_enhanced.toml` - Enhanced colors
- `tokyo_night_storm.toml` - Storm variant

### Rose Pine
All natural pine, faux fur and a bit of soho vibes:
- `rose_pine.toml` - Main variant
- `rose_pine_dawn.toml` - Light variant
- `rose_pine_moon.toml` - Medium variant

### Nightfox
A highly customizable theme family:
- `nightfox.toml` - Dark blue
- `dawnfox.toml` - Light
- `dayfox.toml` - Bright light
- `duskfox.toml` - Dark
- `nordfox.toml` - Nord-inspired
- `terafox.toml` - Green-tinted
- `carbonfox.toml` - High contrast dark

## Usage Example

To switch to a different theme, edit your `~/.config/alacritty/alacritty.toml`:

```toml
# Example: Using Catppuccin Mocha theme
import = [
    "~/.config/alacritty/themes/themes/catppuccin_mocha.toml"
]

# You can also import multiple configuration files
import = [
    "~/.config/alacritty/themes/themes/tokyo_night.toml",
    "~/.config/alacritty/font.toml"
]
```

<Note>
After changing the theme import, Alacritty will automatically reload the configuration. No restart required!
</Note>
